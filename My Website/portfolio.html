Startup Guild

About

The Startup Guild is a website that was built as part of the A100 program. The goal was to design and build a community/jobs website for the startup community. This was a group project and I was put in charge of front-end development. The site was not fully finished as the end goal of the project was not to bring the site live, but to showcase skills.

Features

  • Profile tile wall - tiles flip to reveal info about person or company
  • Sliding navigation - can use the arrows on screen, on keyboard, or swipe gestures
  • Multiple color themes - Users can choose from 6 different color themes
  • Twitter feed - live feed for the website for news

Media Sorter

About

I built this application in C# for myself. You can point it to a directory and give it a destination directory. It will then search for any mp3 files in the directory and will move and sort them in the destination directory. It sorts based on the meta information on the file such as artist and album. It sorts it into a folder structure based on the artists and then the album. If no folder exists for the artist and/or the album one is created. You can save the results of the sort to a text file to keep as a record.

Features

  • Sort and store mp3 files by artists and album
  • Store a log of the changes to keep or review
  • Can be pointed at any directory for the target or destination.

Subway Tetris

About

Subway Tetris is part of my games research. I built this using C# and the Tao Framework for OpenGL. It started as normal Tetris, but at the time I was working at the Subway HQ so I felt Subway was a natural fit for the theme. The artwork is mine as well.

Features

  • All the addictiveness of normal Tetris with a sandwhich flavor
  • Will run on any Windows system with up to date .NET framework.
  • Space bar rotates pieces, arrow keys move, H key holds a piece
