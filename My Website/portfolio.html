3D Tic-Tac-Toe

About

This project I did for my artificial intelligence class. It lets a player play a game of tic-tac-toe against a computer opponent on a 3x3x3 cube. The computer uses alpha beta pruning to make its moves. There are some kinks to work out in the evaluation function. This is something I will continue to work on, so it might be in a different state when you get it.

Features

  • Tic-Tac-Toe on a 3x3x3 cube
  • Computer opponent uses alpha beta pruning to make moves

Media Sorter

About

I built this application in C# for myself. You can point it to a directory and give it a destination directory. It will then search for any mp3 files in the directory and will move and sort them into the destination directory. It sorts based on the meta information on the file such as artist and album. It sorts it into a folder structure based on the artists and then the album. If no folder exists for the artist and/or the album one is created. You can save the results of the sort to a text file to keep as a record.

Features

  • Sort and store mp3 files by artists and album
  • Store a log of the changes to keep or review
  • Can be pointed at any directory for the target or destination.

Subway Tetris

About

Subway Tetris is part of my games research. I built this using C# and the Tao Framework for OpenGL. It started as normal Tetris, but at the time I was working at the Subway HQ so I felt Subway was a natural fit for the theme. The artwork is mine as well.
